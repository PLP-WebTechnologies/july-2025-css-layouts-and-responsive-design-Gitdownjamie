Assignment Write-Up

I started by setting up the structure in HTML with a header, navigation, sidebar, content, and footer. This gave me a clear skeleton to work with.

For the navigation bar, I used Flexbox because it makes it easy to align links in one row. On smaller screens, I used media queries to stack the links vertically so they remain readable.

For the main layout, I chose CSS Grid. On desktop, the grid has two columns (sidebar and content). On tablets and mobile, the layout changes to one column so everything stacks nicely.

Inside the content area, I styled the cards with Flexbox. On desktop, they display in a row, and on mobile, they stack into a column.

I used media queries at 768px (tablet) and 480px (mobile) to adjust fonts, spacing, and layouts. This ensures the design looks good on all devices.

Overall, I combined Grid for the big picture layout and Flexbox for smaller parts like navigation and cards, making the whole site flexible and responsive.
